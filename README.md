## Usage

Use the on-screen buttons to enter numbers and operators.  
Press **=** to calculate, and **Clear** to reset the display.  

### Supported Operations
| Button | Function |
|---------|-----------|

| - | Subtraction |
| * | Multiplication |
| / | Division |
| C | Clear display |
| = | Show result |
